Jak uruchamiać wiele obrazów ISO z jednego USB?

Published Sun, Dec 15, 2013 by Przemek

Estimated reading time: 1 min

Dziś chciałbym przedstawić mój wynik poszukiwań softu do szybkiego robienia flasha z którego można uruchomić wiele obrazów ISO. Testowałem między innymi takie programy jak multibootusb, YUMI i multisystem. Oczywiście wszystko pod xubuntu.

Dla przypomnienia świetnym i naprawdę prostym programem do tworzenia bootowalnego pendrajwa z ISO jest UNetbootin ale może on wgrać tylko jedno ISO co w tym przypadku go eliminuje.

Ostatecznie tylko program Multisystem działał poprawnie i ze stworzonego flasha bez problemu mogłem uruchomić wybrane obrazy ISO.  Bardzo fajną opcją w programie Mulisystem jest odpalanie QEMU, czyli wirtualnej maszyny aby sprawdzić czy flash rzeczywiście jest boot’owalny. Dzięki czemu oszczędzamy sporo czasu i nie trzebą restartować komputera. W moim przypadku na pierwszy ogień poszedł knoppix a potem tails a instrukcja instalacji jest całkiem prosta:

sudo apt-add-repository 'deb http://liveusb.info/multisystem/depot all main'
 wget -q http://liveusb.info/multisystem/depot/multisystem.asc -O- | sudo apt-key add -
 sudo apt-get update
 sudo apt-get install multisystem
